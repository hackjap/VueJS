### Mixsins

- 믹스인은 여러 컴포넌트 간에 공통으로 사용하고 있는 로직, 기능들을 재사용하는 방법
- 믹스인에 정의할 수 있는 재사용 로직은
data, methods, created 등과 같은 컴포넌 트 옵션 

    
        // mixin
        export default {
            // 재사용할 컴포넌트 옵션 & 로직 
        }
        // hoc
        export default  function create(){
            // 재사용할 컴포넌트 옵션
            
        }
